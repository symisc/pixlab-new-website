Generating a UI in React Native

Write a simple React Native app. The app should display a welcome message and a button. When the button is tapped, an alert with the message "Hello!" should appear. Use the SafeAreaView component to ensure proper screen layout and place the text and button in the center of the screen. Style the text with a larger font size (24) and add some space below it. Use React Native's Button component for the button.

Code Generated

import React from 'react';
import { SafeAreaView, Text, Button, StyleSheet } from 'react-native';

const App = () => (
<SafeAreaView style={styles.container}>
<Text style={styles.title}>Welcome to React Native!</Text>
<Button title="Click Me" onPress={() => alert('Hello!')} />
</SafeAreaView>
);

const styles = StyleSheet.create({
container: { flex: 1, justifyContent: 'center', alignItems: 'center' },
title: { fontSize: 24, marginBottom: 20 },
});

export default App;
