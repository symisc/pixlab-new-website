Generate Images with Text Prompts

Transform your ideas into visuals by using descriptive text prompts directly in the URL. This functionality allows seamless integration with other PixLab transformations, enabling you to create unique assets on demand, effortlessly and at scale.

// Sample fetch request to generate an image using PixLab's Media Processing API
    const apiKey = "YOUR_API_KEY";
    const prompt = "sunset over mountains in watercolor style";
    const url = `https://api.pixlab.io/media/process?api_key=${apiKey}&prompt=${encodeURIComponent(prompt)}&format=png`;
    
    fetch(url)
      .then((response) => {
        if (!response.ok) {
          throw new Error("Failed to process the image");
        }
        return response.json();
      })
      .then((data) => {
        console.log("Generated image URL:", data.image_url);
      })
      .catch((error) => console.error("Error:", error));
