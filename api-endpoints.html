REST API ENDPOINTS

This page lists all the exposed PixLab API endpoints. As of this release, there are over 150 API endpoints covering Computer Vision, Generative AI, Large Language Models (LLMs), and Media Processing tasks, with enhancements and new endpoints added in each release. This is the API endpoints reference document to follow. If you have any suggestions or feature requests, please open a new ticket via the support tab of the PixLab Console ↗. For a tutorial introduction, refer to Getting Started with PixLab and The REST-APIs Code Samples. Feel free to copy, paste, and run these examples to see the API in action.

Text Inpainting

Ever wanted to replace a object in a picture, or add something nice in the background. Based on artificial intelligence, this API enables you to do inpainting and update part of an image describing what you want instead.

Documentation and code samples

Uncrop

Uncrop your pictures to any format, extend bodies, objects, or sceneries to enlarge your perspective and get your picture with just the right amount of context.

Documentation and code samples

Remove Background

With the Clipdrop background removal API, you can quickly remove the background of any image with incredible accuracy without having to do a bunch of extra work! It's the most accurate background removal solution available on the market.

Documentation and code samples

Image upscaling

Sometimes the only thing preventing you from displaying a memorable picture is its low resolution. With the image upscaling API, transform your low resolution image into an ultra sharp high resolution image. Leaning on advanced computer vision algorithms you can upscale your up to x16.

Documentation and code samples

Cleanup

Too many photos contain unwanted objects, text, or persons. Now, you can simply remove all these flaws with Cleanup API. Based on artificial intelligence, this API enables you to clean and remove flaws from photographs, such as strangers in the background or blemishes on portraits.

Documentation and code samples

Reimagine

Reimagine and view your images differently with the Reimagine API. Based on your input image, we leverage the latest Stable Diffusion tech to create an infinite number of variations to suit all your use cases!
